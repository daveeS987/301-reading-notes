### Jupyter

- "-" "=" Zoom in/ Zoom Out
- "[" "]" Rotate Left/Right
- 0 Reset

Command Mode

- a - add cell above
- b - add cell below
- c - copy
- v - paste
- x - cut
- dd - delete cell
- z - undo
- shift z - redo
- y - cell format code
- m - markdown

Edit Mode

- enter - edit currently active cell
- shift enter - run cell
- 00 - restart kernel
